postcss-button

Github

A PostCSS plugin to create buttons.

Settings

@button [name] {
  color: grey;
  background-color: white;
  color-active: white;
  background-color-active: grey;
  border-width: 0;
  border-style: none;
  border-color: grey;
  border-color-active: black;
  class-active: active;
  class-disabled: disabled;
}

Buttons

.my-button { button: [name] }
